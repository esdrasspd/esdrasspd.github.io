Mis Proyectos


**Todos estos proyectos han sido vendidos a diferentes clientes y empresas/negocios**



Formain (Aplicación Desktop)

Este programa es utilizado por 3 empresas, permite gestionar y administrar la empresa o negocio. Está desarrollado totalmente en Java y SQL.



Servicios Contables Ola (Aplicación Desktop)

Programa creado para la oficina contable "Ola" ubicado frente al parque central de San Antonio Suchitepéquez. La funcionalidad de este programa es que dicho negocio pueda llevar los datos de sus clientes y controlar las fechas de pago. Este programa fue desarrollado en Java y SQL.



Portafolio Propio

Página web responsiva (esta página) para dar a conocer mis proyectos de programación y que me puedan contactar. Las tecnologías utilizadas son: HTML, CSS y Javascript.



Supercampeones

Este programa fue creado para llevar el registro y control de varios equipos de fútbol, y saber cuándo se van a enfrentar. Las tecnologías utilizadas son: C++, .NET y SQL.



Control Menu

Esta aplicación móvil fue creada para un comedor ubicado en Mazatenango, con el fin de registrar los platillos que preparan y saber los precios. Las tecnologías utilizadas son: Dart, Flutter, SQLite, gestor de estado BloC_Provider.



Movies Info

Esta aplicación móvil fue creada para ver las películas populares en la actualidad. Al entrar a una cartelera muestra información de dicha película y los actores. Las tecnologías utilizadas son: Dart, Flutter, manejo de API de MovieDB, gestor de estado BloC_Provider.



Conociendo Guate

Esta aplicación móvil actualmente se encuentra en desarrollo, servirá para dar a conocer a los extranjeros los lugares y municipios bonitos de Guatemala. . Las tecnologías utilizadas son: Dart, Flutter, API propia en desarrollo, gestor de estado BloC.
